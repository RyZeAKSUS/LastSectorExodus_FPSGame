## Descrição do Jogo

**Last Sector: Exodus** é um jogo de tiro em primeira pessoa (FPS) em que o jogador enfrenta hordas de inimigos com dificuldade crescente. O objetivo é sobreviver o maior número de ondas possível, eliminando todos os inimigos antes que eles o eliminem. À medida que o jogo avança, as hordas tornam-se maiores e mais desafiadoras, exigindo boa gestão de munição, equipamento e posicionamento.

---

## Funcionalidades Implementadas

- Movimentação completa do jogador (andar, correr, agachar, saltar)
- Sistema de combate com disparo, mira e recarga
- Alternância de modo de disparo
- Sistema de inventário com itens utilizaveis (teclas 1–9 e scroll)
- Recolha de itens e munição no ambiente
- Sistema de hordas com dificuldade progressiva
- Menu de pausa
- Sistema de fome, sede, adrenalina e afogamento

## Jogabilidade

### Objetivo
Sobreviver ao maior número de hordas possível. Cada horda traz mais inimigos e maior dificuldade. O jogo termina quando o jogador é derrotado.

## Como Abrir o Projeto

1. Clona ou faz download deste repositório para o teu computador.
2. Abre o **Unity Hub** e clica em **"Add"** (ou **"Adicionar projeto"**).
3. Seleciona a pasta raiz do projeto (onde está a pasta `Assets`).
4. O projeto aparecerá na lista — clica nele para abrir.
5. Dentro do Unity Editor, abre a cena principal em `Assets/Map/Scenes/Map(nao pasta)`.
6. Carrega no botão **Play** para correr o jogo.
